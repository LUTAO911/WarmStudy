## 本项目说明

本项目在自主完成系统方案设计、业务逻辑实现、前后端页面开发、智能体流程编排、心理陪伴场景设计与系统集成的基础上，使用了若干开源框架、开源库及第三方平台 SDK，用于实现后端服务构建、RAG 检索、向量存储、OCR、监控与接口调用等能力。

## 主要开源组件

1. Flask / Flask-CORS
2. LangChain 相关组件
3. ChromaDB
4. Pydantic
5. Requests / Aiohttp
6. Redis / Aioredis
7. Prometheus Client / Psutil
8. PyMuPDF / Pillow / Pytesseract
9. 微信小程序原生框架
