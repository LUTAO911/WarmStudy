## 当前实现特点

- 当前版本主要使用 API 调用方式，而非随仓库打包本地模型权重
- 通过环境变量配置模型名称与 API Key
- 适合比赛演示与快速部署

## 当前所需环境变量

```env
CHAT_MODEL=qwen
DASHSCOPE_API_KEY=your_dashscope_api_key
DASHSCOPE_MODEL=qwen-plus
```
