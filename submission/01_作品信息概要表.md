# 作品信息概要表

## 作品名称

WarmStudy（暖学帮）- 青少年心理关怀智能体系统

## 参赛方向

教育方向

## 细分场景

助育与青少年心理支持

## 项目背景

青少年心理健康问题在学校与家庭场景中日益突出。学生缺少低门槛、可持续的情绪表达通道，家长难以及时感知情绪趋势，学校也缺少轻量化、可数字化的心理支持工具。基于这一痛点，本项目设计并实现了一个面向青少年心理关怀的智能体系统原型。

## 核心问题

本项目主要解决以下问题：

1. 学生缺少便捷、连续、低压力的情绪表达入口。
2. 家长难以理解孩子的长期情绪变化趋势，也缺少结构化沟通建议。
3. 学校缺少用于早发现、轻跟踪、弱预警的数字化心理支持工具。

## 项目定位

本项目定位为面向学校与家庭场景的青少年心理关怀 AI 助手平台，不作为诊断或治疗工具，而是作为低门槛心理支持与教育辅助系统。

## 核心功能

1. 学生端心理陪伴对话
2. 每日情绪打卡与状态记录
3. 家长端 AI 对话与趋势查看
4. 基于心理知识库的 RAG 检索增强
5. 风险线索识别与预警支持
6. 报告与跟踪信息支持

## 技术路线

项目采用“大模型 + RAG + 工具调用 + 前后端一体化原型”的技术路线。

- 前端：微信小程序
- 后端：Python 服务架构
- 智能体能力：LangChain 相关组件
- 向量数据库：ChromaDB
- 模型接入：通义千问 / DashScope API、MiniMax API

## 创新亮点

1. 聚焦教育方向下的青少年心理关怀，而不是通用问答场景
2. 构建学生、家长、学校三方支持闭环
3. 将心理陪伴、知识检索、风险识别整合到统一智能体原型中
4. 兼顾比赛演示可交付性与后续商业落地可扩展性

## 部署形式

当前版本支持本地部署或云服务器部署，比赛演示阶段推荐采用 API 方式调用大模型，以保证部署效率与演示稳定性。

## 项目价值

1. 降低青少年获得情绪支持的门槛
2. 帮助家长更早识别情绪变化趋势
3. 为学校提供轻量化心理关怀数字工具
4. 体现大模型与智能体技术在教育场景中的应用价值
