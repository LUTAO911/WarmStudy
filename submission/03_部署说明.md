# 部署说明

正式部署请以 [docs/DEPLOYMENT_RUNBOOK.md](/C:/Users/34206/OneDrive/Desktop/WarmStudy-main/docs/DEPLOYMENT_RUNBOOK.md) 为准。

## 当前部署形态

- 正式接口域名：`https://wsapi.supermoxi.top`
- 对外唯一网页入口：`8000`
- 内部 Agent / RAG API：`5177`
- `5177` 不再单独对外提供 Web UI

## 推荐部署方式

1. 服务器使用 Docker 部署
2. 只暴露 `8000`
3. Nginx 或 Caddy 反向代理到 `8000`
4. `5177` 仅保留容器内或内网访问

## 关键环境变量

```env
CHAT_MODEL=qwen
DASHSCOPE_API_KEY=your_dashscope_api_key
DASHSCOPE_MODEL=qwen-plus
AGENT_API_KEY=your_admin_api_key
RAG_AGENT_URL=http://localhost:5177
FLASK_ENV=production
LOG_LEVEL=INFO
```

## 现场演示建议

1. 提前验证管理员后台 `8000` 可打开
2. 提前验证上传文件、RAG 检索和问答功能
3. 提前验证模型 Key 可用
4. 准备一份演示视频作为网络异常备份

## 家长-学生绑定演示步骤

1. 学生端手机号登录，系统生成固定的 9 位孩子 ID。
2. 学生进入“成长中心”，复制孩子 ID。
3. 家长端手机号登录；如果尚未绑定孩子，首页会先显示绑定引导。
4. 家长输入 9 位孩子 ID 完成绑定。
5. 学生完成测评或打卡后，家长端刷新即可查看综合报告、测评报告与预警。
