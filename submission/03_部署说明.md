# 部署说明

正式部署请以 [docs/DEPLOYMENT_RUNBOOK.md](/C:/Users/34206/OneDrive/Desktop/WarmStudy-main/docs/DEPLOYMENT_RUNBOOK.md) 为准。

## 当前部署形态

- 正式接口域名：`https://wsapi.supermoxi.top`
- 对外唯一网页入口：`8000`
- 内部 Agent / RAG API：`5177`
- `5177` 不再单独对外提供 Web UI

## 推荐部署方式

1. 服务器使用 Docker 部署
2. 只暴露 `8000`
3. Nginx 或 Caddy 反向代理到 `8000`
4. `5177` 仅保留容器内或内网访问

## 关键环境变量

```env
CHAT_MODEL=qwen
DASHSCOPE_API_KEY=your_dashscope_api_key
DASHSCOPE_MODEL=qwen-plus
RAG_DASHSCOPE_MODEL=qwen-plus
DASHSCOPE_EMBEDDING_MODEL=text-embedding-v3
DASHSCOPE_EMBEDDING_FALLBACK_MODEL=text-embedding-v2
AGENT_API_KEY=your_admin_api_key
RAG_AGENT_URL=http://localhost:5177
FLASK_ENV=production
LOG_LEVEL=INFO
```

## 真机通信要求

安装包真机访问依赖腾讯 miniapp / super app 后台通信域名配置。比赛前需确认以下域名已配置，并在配置后重新打包安装：

- request 合法域名：`https://wsapi.supermoxi.top`
- uploadFile 合法域名：`https://wsapi.supermoxi.top`
- downloadFile 合法域名：`https://wsapi.supermoxi.top`
- websocket 合法域名：如启用长连接，配置 `wss://wsapi.supermoxi.top`
