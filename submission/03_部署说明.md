# 部署说明

正式部署说明请以以下文档为准：

- `docs/DEPLOYMENT_RUNBOOK.md`

## 当前项目结构

- 前端目录：`WarnStudty`
- 后端目录：`agent`
- Agent / RAG 服务端口：`5177`
- API Gateway 端口：`8000`

## 推荐比赛演示部署方式

1. 在本地或云服务器中部署 Python 后端
2. 通过 `.env` 配置模型 API Key
3. 启动 `agent/app.py` 与 `agent/api_gateway.py`
4. 在微信开发者工具中导入 `WarnStudty`
5. 将前端 API 地址指向 `http://localhost:8000`

## 关键环境变量

```env
CHAT_MODEL=qwen
DASHSCOPE_API_KEY=your_dashscope_api_key
DASHSCOPE_MODEL=qwen-plus
MINIMAX_API_KEY=your_minimax_api_key
RAG_AGENT_URL=http://localhost:5177
FLASK_ENV=production
LOG_LEVEL=INFO
```
