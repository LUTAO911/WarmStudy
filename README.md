## 一、项目概述

### 1.1 项目背景

青少年心理健康问题已成为社会关注的焦点。《中国青年发展报告》显示，我国约3000万青少年存在不同程度的心理困扰。然而，专业心理咨询资源稀缺，学生、家长、教师三方都缺乏有效的心理支持工具。

**暖学帮**正是为解决这一痛点而设计——基于大语言模型构建智能心理关怀系统，为青少年提供触手可及的心理支持。

### 1.2 核心定位

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   暖学帮 = 大语言模型 + RAG知识库 + 工具调用                │
│                                                              │
│   定位：青少年心理关怀AI                                      │
│   使命：让每一位青少年都能获得温暖、专业的心理支持            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 项目特色

| 特色 | 说明 |
|------|------|
| 🎯 **领域专注** | 区别于通用教育助手，聚焦青少年心理健康 |
| 🔬 **循证方法** | 基于认知行为疗法(CBT)等心理学原理 |
| 🛡️ **安全边界** | 三级危机预警 + 专业转介机制 |
| 👨‍👩‍👧 **三方闭环** | 学生-家长-教师心理支持闭环 |

## 五、快速部署

### 5.1 环境要求

| 组件 | 最低 | 推荐 |
|------|------|------|
| Python | 3.10 | 3.11+ |
| Docker | 24.0 | 24.0+ |
| 内存 | 8GB | 16GB+ |

### 5.2 一键安装 (Windows)

双击运行 `agent/setup.bat`，会自动：
1. 检测 Python 版本
2. 创建虚拟环境 (.venv)
3. 安装所有依赖
4. 生成 .env 配置文件

### 5.3 手动安装

```bash
# 1. 进入 agent 目录
cd agent

# 2. 创建虚拟环境
python -m venv .venv

# 3. 激活虚拟环境 (Windows)
.venv\Scripts\activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 配置环境变量
copy .env.example .env
# 编辑 .env，填入你的 DASHSCOPE_API_KEY
```

### 5.4 启动服务

```bash
# 方式1: 使用脚本 (Windows)
start_venv.bat

# 方式2: 手动启动
.venv\Scripts\activate
python app.py

# 方式3: Docker部署
docker-compose up -d
```

### 5.5 验证

启动后访问: http://localhost:5177/api/docs
