# 危机识别与干预指南

## 什么是心理危机？

心理危机是指个体在遇到突发事件或生活压力时，原本的心理平衡被打破，出现情绪崩溃、行为失控的状态，可能对自己或他人造成伤害。

## 需要警惕的信号

### 言语信号
- 直接表达："我想死"、"我不想活了"
- 间接表达："活着有什么意思"、"我太累了"
- 与家人朋友告别："谢谢你们对我的照顾"
- 谈论自伤的具体方法
- 撰写遗书或告别性质的内容

### 行为信号
- 突然的社交退缩（不愿见人、不想说话）
- 行为明显改变（从活跃变得沉默，或反之）
- 睡眠问题（长期失眠或过度嗜睡）
- 食欲明显改变
- 把珍贵物品送人
- 酗酒或吸毒
- 自伤行为（割腕等）

### 情绪信号
- 长期情绪低落
- 强烈的愧疚感和无价值感
- 对一切失去兴趣
- 易怒、暴躁
- 绝望感

### 生理变化
- 体重骤降
- 面色憔悴
- 眼神无光
- 个人卫生变差

## 如果你发现自己有以上信号

### 首先，记住这三件事：
1. **你现在的感受是真实的，但它们会改变**
2. **帮助就在身边，你不需要独自承受**
3. **寻求帮助是勇敢的表现，不是软弱**

### 可以做的事情：
- 立即告诉信任的人（家长、老师、朋友）
- 联系学校心理老师
- 拨打心理援助热线
- 去正规医院心理科就诊

## 如果你发现朋友有危机信号

### 你可以：
1. **直接询问**：真诚地表达关心，如"我注意到你最近好像不太开心，你还好吗？"
2. **认真倾听**：放下手机，全身心地倾听对方
3. **不要评判**：不要批评或指责对方
4. **鼓励求助**：鼓励朋友寻求专业帮助
5. **陪伴左右**：如果情况紧急，保持陪伴，并通知成年人

### 你不应该：
- 假装没看见
- 答应保密（安全问题是例外）
- 嘲笑或轻视对方的感受
- 独自承担帮助的责任

## 危机干预流程

```
识别信号 → 建立连接 → 倾听理解 → 评估风险 → 寻求帮助 → 持续支持
```

## 记住

无论你现在多么绝望，请相信：黑暗不会永远持续，总会有光明在等着你。你不是一个人，有人在乎你，有人在帮助你。

如果你有自伤的想法，请立刻：
1. 告诉身边的人
2. 联系心理援助热线
3. 去最近的医院急诊

你值得被爱，你值得拥有美好的未来。💕
