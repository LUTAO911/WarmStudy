# WarmStudy智能心理辅导系统使用说明书

## 1. 系统概述

### 1.1 什么是WarmStudy？
WarmStudy是一个基于人工智能技术的智能心理辅导系统，专为青少年和家长设计，提供24/7的心理健康支持服务。

### 1.2 为什么产生WarmStudy？
- **青少年心理健康问题日益严重**：现代社会竞争压力大，青少年面临学业、人际关系等多方面压力
- **传统心理辅导资源不足**：专业心理咨询师数量有限，难以覆盖所有需要帮助的青少年
- **隐私和便利性需求**：青少年对传统心理咨询存在抵触情绪，更倾向于使用数字化工具
- **家长缺乏专业指导**：家长普遍缺乏心理健康知识，需要专业支持来帮助孩子

### 1.3 WarmStudy有什么用？
- **智能心理辅导**：通过AI对话为青少年提供心理支持
- **情绪识别与干预**：实时监测情绪状态，及时发现危机信号
- **专业知识解答**：基于心理健康知识库提供科学的建议
- **家长协同支持**：家长端查看孩子心理状态，获取专业指导
- **个性化服务**：根据孩子的年龄、性别等信息提供定制化的心理支持

## 2. 系统架构

### 2.1 技术架构
- **前端**：微信小程序（学生端和家长端）
- **后端**：FastAPI服务
- **智能体**：基于LangChain框架的AI智能体
- **知识库**：RAG增强的心理健康知识库
- **大模型**：通义千问和MiniMax模型

### 2.2 核心功能模块
- **智能对话模块**：处理用户输入，生成心理辅导回复
- **知识检索模块**：从知识库中检索相关专业知识
- **情绪分析模块**：识别用户情绪状态，检测危机信号
- **记忆管理模块**：存储用户偏好和历史对话
- **家长监控模块**：家长端查看孩子心理状态

## 3. 快速开始

### 3.1 系统要求
- **前端**：微信小程序（支持Android和iOS）
- **后端**：支持Python 3.10+的服务器
- **网络**：稳定的网络连接
- **API密钥**：通义千问和MiniMax API密钥

### 3.2 安装部署
1. **后端部署**：
   - 克隆项目代码到服务器
   - 配置环境变量（参考`.env.example`）
   - 运行`docker-compose up -d`启动服务

2. **前端配置**：
   - 在微信开发者工具中打开`WarmStudy`项目
   - 修改`utils/api.js`中的API地址为后端服务地址
   - 编译并发布小程序

### 3.3 首次使用流程
1. **扫码登录**：扫描小程序码进入登录页面
2. **身份选择**：选择"学生"或"家长"身份
3. **信息填写**：
   - 学生端：填写姓名、性别、年龄、年级
   - 家长端：填写联系方式，绑定孩子信息
4. **开始使用**：进入主界面，开始与AI助手聊天

## 4. 学生端使用指南

### 4.1 智能对话
**功能说明**：与AI心理辅导助手"暖暖"进行实时对话

**使用步骤**：
1. 进入"聊天"页面
2. 在输入框中输入你的问题或心事
3. 点击发送按钮
4. 等待AI助手的回复
5. 可以继续追问或开启新的话题

**示例对话**：
- 你：今天考试没考好，很沮丧
- 暖暖：我理解你的感受，考试失利确实会让人难过。能告诉我更多关于这次考试的情况吗？
- 你：我复习了很久，但还是没考好
- 暖暖：你已经很努力了，这很棒！考试成绩不代表你的全部价值，我们可以一起分析原因，找到改进的方法。

### 4.2 心理评估
**功能说明**：完成心理健康评估问卷，了解自己的心理状态

**使用步骤**：
1. 进入"评估"页面
2. 点击"开始评估"按钮
3. 认真回答问卷中的问题
4. 完成后查看评估结果
5. 系统会根据评估结果提供相应的建议

### 4.3 资源库
**功能说明**：浏览心理健康相关的文章和资源

**使用步骤**：
1. 进入"资源"页面
2. 浏览推荐的心理健康文章
3. 点击文章查看详细内容
4. 可以收藏感兴趣的文章

## 5. 家长端使用指南

### 5.1 孩子状态监控
**功能说明**：查看孩子的心理状态和对话历史

**使用步骤**：
1. 进入"首页"页面
2. 查看孩子的最近心理状态
3. 点击"查看详情"查看具体信息
4. 可以看到孩子的情绪趋势和高频话题

### 5.2 家长AI对话
**功能说明**：向AI助手咨询关于孩子教育和心理健康的问题

**使用步骤**：
1. 进入"AI对话"页面
2. 输入你关于孩子教育的问题
3. 点击发送按钮
4. 查看AI助手的专业建议

**示例问题**：
- 我的孩子最近总是情绪低落，我该怎么办？
- 如何帮助孩子缓解考试压力？
- 孩子不愿意和我沟通，应该怎么处理？

### 5.3 告警通知
**功能说明**：当系统检测到孩子情绪异常时，及时向家长发送告警

**使用步骤**：
1. 进入"告警"页面
2. 查看系统发送的告警信息
3. 点击"查看详情"了解具体情况
4. 按照系统建议采取相应措施

## 6. 知识库管理

### 6.1 Web管理界面
**功能说明**：通过Web界面管理心理健康知识库

**访问方式**：浏览器访问 `http://localhost:8501`

**主要功能**：
- **文件上传**：上传各种格式的知识文件（md、txt、pdf、docx等）
- **知识搜索**：搜索知识库中的内容
- **分类管理**：按分类管理知识内容
- **统计信息**：查看知识库的统计数据

### 6.2 支持的文件格式
- **文本文件**：.md, .txt
- **文档文件**：.pdf, .docx, .doc
- **表格文件**：.xlsx, .csv
- **数据文件**：.json
- **网页文件**：.html, .htm

### 6.3 知识分类
- **心理健康基础知识**：心理健康的基本概念和原理
- **情绪管理**：情绪识别和调节方法
- **压力管理**：应对压力的策略
- **人际关系**：社交技能和关系处理
- **学习适应**：学习方法和考试焦虑
- **危机干预**：自杀预防和危机处理

## 7. 高级功能

### 7.1 个性化设置
**功能说明**：根据孩子的特点定制AI助手的回应风格

**设置项**：
- **沟通风格**：友好、专业、幽默等
- **回应长度**：简短、中等、详细
- **话题偏好**：学习、生活、兴趣等
- **隐私设置**：数据存储和使用偏好

### 7.2 语音输入
**功能说明**：支持语音输入，方便不方便打字的场景

**使用方法**：
1. 点击输入框旁边的麦克风图标
2. 说出你想表达的内容
3. 系统会自动识别并转换为文本
4. 确认无误后点击发送

### 7.3 多语言支持
**功能说明**：支持中英文切换，满足不同语言需求

**切换方法**：
1. 进入"设置"页面
2. 选择"语言设置"
3. 选择你需要的语言
4. 系统会自动切换界面语言

## 8. 常见问题

### 8.1 技术问题
**Q: 系统响应缓慢怎么办？**
A: 可能是网络问题或服务器负载过高。请检查网络连接，或稍后再试。

**Q: 上传文件失败怎么办？**
A: 请检查文件格式是否支持，文件大小是否超过限制（建议不超过10MB）。

**Q: 登录失败怎么办？**
A: 请检查网络连接，或尝试重新登录。如果问题持续，请联系管理员。

### 8.2 使用问题
**Q: AI助手的回复不够准确怎么办？**
A: 可以尝试更详细地描述你的问题，或使用更具体的关键词。系统会不断学习和改进。

**Q: 如何保护孩子的隐私？**
A: 系统采用数据加密存储，所有对话内容仅用于提供服务，不会用于其他目的。家长可以在设置中查看隐私政策。

**Q: 家长如何查看孩子的对话历史？**
A: 家长端首页会显示孩子的心理状态概览，点击"查看详情"可以查看具体的对话历史和情绪分析。

### 8.3 其他问题
**Q: 系统可以识别哪些情绪？**
A: 系统可以识别焦虑、抑郁、愤怒、恐惧等多种情绪状态。

**Q: 如何更新知识库？**
A: 管理员可以通过Web管理界面上传新的知识文件，系统会自动处理并添加到知识库中。

**Q: 系统适合什么年龄段的孩子？**
A: 系统主要面向6-18岁的青少年，针对不同年龄段会提供相应的心理支持。

## 9. 系统维护

### 9.1 后端服务维护
- **日志查看**：通过`docker-compose logs`查看服务日志
- **服务重启**：`docker-compose restart`重启服务
- **更新部署**：`docker-compose up -d --build`重新构建并启动

### 9.2 知识库维护
- **定期更新**：定期添加新的心理健康知识
- **知识审核**：确保知识库内容的准确性和专业性
- **性能优化**：定期清理过期数据，优化检索性能

### 9.3 安全维护
- **API密钥管理**：定期更新API密钥，避免泄露
- **数据备份**：定期备份数据库和知识库
- **安全检查**：定期检查系统安全漏洞
