# WarmStudy智能心理辅导系统 - 部署说明

## 1. 部署环境要求

### 1.1 硬件要求
| 配置 | 最低要求 | 推荐配置 |
|------|----------|----------|
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 磁盘 | 50GB | 100GB |
| 网络 | 1Mbps | 5Mbps |

### 1.2 软件要求
- **操作系统**：Windows 10/11、Linux (Ubuntu 20.04+)、macOS
- **Docker**：Docker 20.10+、Docker Compose 1.29+
- **Python**：3.10+（仅本地开发需要）
- **微信开发者工具**：用于前端开发和预览

## 2. 部署方式

### 2.1 Docker 容器化部署（推荐）

#### 步骤 1：准备项目
1. 克隆项目代码：
   ```bash
   git clone https://github.com/your-repo/nuanxuebang.git
   cd nuanxuebang
   ```

2. 配置环境变量：
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，填写 API 密钥
   ```

#### 步骤 2：启动服务
```bash
# 构建并启动服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f
```

#### 步骤 3：验证服务
- **后端 API**：访问 http://localhost:8000/docs
- **健康检查**：访问 http://localhost:8000/health
- **RAG 管理界面**：访问 http://localhost:8501

### 2.2 本地开发部署

#### 步骤 1：安装依赖
```bash
# 后端依赖
cd backend
pip install -r requirements.txt

# 前端依赖（微信小程序无需额外安装）
```

#### 步骤 2：启动服务
```bash
# 启动后端服务
cd backend
python main.py

# 启动 RAG 管理界面
cd backend
streamlit run web/rag_admin.py
```

## 3. 环境变量配置

### 3.1 核心环境变量
| 变量 | 说明 | 示例 | 必需 |
|------|------|------|------|
| MINIMAX_API_KEY | MiniMax API 密钥 | sk-cp-xxx | 是 |
| QWEN_API_KEY | 通义千问 API 密钥 | sk-xxx | 是 |
| DATABASE_URL | 数据库连接地址 | sqlite:///./data/nuanxuebang.db | 否 |
| VECTOR_DB_PATH | 向量数据库路径 | ./vector_db | 否 |
| KNOWLEDGE_BASE_PATH | 知识库路径 | ./knowledge | 否 |

### 3.2 高级配置
| 变量 | 说明 | 默认值 |
|------|------|--------|
| PORT | 后端服务端口 | 8000 |
| HOST | 后端服务主机 | 0.0.0.0 |
| DEBUG | 调试模式 | False |
| CORS_ORIGINS | CORS 允许的源 | * |

## 11. 故障排查

### 11.1 常见问题

**问题 1：服务启动失败**
- **症状**：`docker-compose up -d` 后服务状态为 `Exited`
- **原因**：可能是环境变量配置错误或端口冲突
- **解决**：
  ```bash
  # 查看详细日志
  docker-compose logs backend
  
  # 检查端口是否被占用
  netstat -ano | findstr :8000
  ```

**问题 2：API 调用失败**
- **症状**：前端调用API返回错误
- **原因**：可能是网络问题或API密钥错误
- **解决**：
  ```bash
  # 检查后端服务状态
  docker-compose ps
  
  # 检查API密钥配置
  cat .env
  ```

**问题 3：知识库上传失败**
- **症状**：上传文件时显示错误
- **原因**：可能是文件格式不支持或文件过大
- **解决**：
  - 检查文件格式是否在支持列表中
  - 确保文件大小不超过10MB
