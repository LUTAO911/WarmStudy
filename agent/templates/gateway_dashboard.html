产品亮点与智能体设计

把后台数据、心理陪伴、家长支持和 RAG 知识库真正串成一个会工作的场景智能体。

实时理解 App 状态

学生一旦提交打卡、测评或触发预警，后台状态会立刻刷新。智能体不只看当前一句话，而是能结合最近的心理状态、风险变化和会话上下文调整回应。

即时状态联动 测评/打卡入会话 支持预警感知

学生与家长双人格

学生端优先共情、适龄表达、避免说教；家长端优先结构化建议、观察点和下一步动作。两端不再共用同一套提示词身份，减少答非所问和角色错位。

学生陪伴型 家长支持型 角色分流

比普通聊天机器人更“懂场景”

WarmStudy 的巧妙之处不只是调用大模型，而是把 RAG 知识、用户资料、测评状态、管理后台和对话策略放进同一个闭环，形成可观测、可管理、可迭代的校园心理支持系统。
